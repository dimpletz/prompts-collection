# Leader `v1.0.0`

> A collection of agents for creating compelling presentations and communicating ideas effectively.

## Prerequisites

- [VS Code](https://code.visualstudio.com/) with the [GitHub Copilot Chat](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot-chat) extension installed and active.

## Installation

Install via the VS Code Chat Plugin Marketplace using the `dimpletz/prompts-collection` marketplace source and enable the **leader** plugin.

## Usage

Open Copilot Chat, select the **Presenter** agent, then tell it what topic you need a presentation on.

```
# Examples
"Create a presentation on microservices architecture"
"Build a 10-slide deck about the benefits of DevOps"
"Generate a presentation introducing our new product roadmap"
```

> The Presenter agent responds **only** to requests for presentation creation or improvement. For any other task, use a different plugin.

## Components

### Presenter

An expert presentation creator that generates comprehensive, engaging, and well-structured presentations on any topic. The agent:

1. Conducts automated web research to gather current and accurate information on the topic.
2. Structures content into a clear, professional format with compelling narratives.
3. Iterates until the presentation is complete and polished.

**Capabilities:**
- Any topic — technical, business, or strategic
- Research-backed, up-to-date content
- Clear slide structure with titles, key points, and speaker notes
- Autonomous operation until delivery is complete
